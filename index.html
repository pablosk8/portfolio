Bienvenido

¡Hola, soy Pablo, desarrollador de Front-End! Mi enfoque es la creación de experiencias web atractivas y funcionales. Trabajo con HTML y CSS para dar vida a diseños y garantizar que los sitios se vean y funcionen de manera excelente en todos los dispositivos. Estoy aquí para hablar sobre mi trabajo y cómo el diseño web puede impactar positivamente en su proyecto.

Conocimientos

HTML es un lenguaje de marcado utilizado para crear y diseñar páginas web.

CSS es un lenguaje de estilo para controlar la presentación y el diseño de HTML.

JavaScript se utiliza principalmente para agregar interactividad y funcionalidad a las páginas web.
